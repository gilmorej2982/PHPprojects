Writing a simple form

Writing a form and using a script to process the input

Write a simple form that can take user input and process it. You'll need the following form elements:

  • The form itself. Don't forget the required method and action attributes. The action needs to connect to the script you'll write on form-script.php.
  • You'll have 3 sections to the form: Personal Information, Comments and Submit Your Form. Use fieldset with its legend to separate the three sections.
  • In Personal Information, get the user's First Name and Last Name. You'll use input fields for these. Remember the type, name and id attributes for input fields. Name and id usually match, except for checkboxes (which we are not doing today).
  • In Comments, use a textarea so the user can input some text. Use name and id attributes.
  • For your input and textareas, use label to label them. This is an important usability feature. Labels come *before* the input and text areas so that assistive technologies read them in a logical way for people who cannot see them. In addition to that, use the for attribute in each label and be sure it matches the name/id of the field.
  • Use a submit type of input in the last fieldset so the form can be submitted. This is what allows the form to send the data to the script you've specified in the action attribute. The special submit input field can take the value attribute to customize the text on the button itself. Keep it simple.

Don't worry about styling the form. You will do that as homework.
